# smartshop-online-shopping
Discover our range of premium smart devices, including cutting-edge phones for modern connectivity, powerful laptops for enhanced productivity, crystal-clear noise-cancelling earbuds, and rugged, portable Bluetooth speakers for on-the-go audio

# Installation:
   Download the Zip file or Clone the repositary  to your local machine

# Dependencies:
   Ensure you have Installed Python new version

   Install below version in terminal and 'New Version will face version conflict error'
   ```
      pip install django
      pip install pillow
      pip install django-jazzmin
  ```
# Run the code:
After installation Run this code

   ```
       python manage.py runserver
       
   ```
